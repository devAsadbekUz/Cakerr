# AI Coding Assistant Memories

This file contains preferences, constraints, and persistent rules for the AI coding assistant.

## General Preferences
- **No Automatic Browser Previews**: The user prefers to perform all verification and checking of updates manually. Do NOT use the browser subagent or any automated tools to verify UI changes or functionality unless explicitly asked.
- **Verification Responsibility**: Always wait for the user to confirm that a change is working as intended in their local environment.
